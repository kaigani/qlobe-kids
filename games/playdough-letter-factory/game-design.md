# Playdough Letter Factory - Game Design

### Mini-GDD: Playdough Letter Factory (games/playdough-letter-factory/)
**Category:** writing-fine-motor · **Engine:** build-assemble · **splashEmoji:** 🟣
**Premise:** roll and place playdough snakes and curves onto the letter ghost to build squishy letters. The voice names each part, then celebrates the completed form and invites real playdough play.
**Learning:** letter construction from strokes and parts-of-letters thinking.
**Modes:** 1. `letters` "Squish a Letter" (rounds 4): L, O, T, B with ordered dough-part placement inside each build. 2. `numbers` "Squish a Number" (rounds 3): 1, 7, 0 with ordered dough-part placement inside each build.
**Assets needed:** playdough snake/ball/curve art with squish textures; recorded voice.
