# Turn-Taking Tower - Game Design

### Mini-GDD: Turn-Taking Tower (games/turn-taking-tower/)
**Category:** social-emotional · **Engine:** build-assemble · **splashEmoji:** 🏗️
**Premise:** build the tower together. The current beta uses ordered block
placement plus spoken narration such as "Maya places hers... now it is your
turn" to approximate friend turns while staying inside the shared engine.
**Learning:** turn-taking rhythm, patience, shared building pride.
**Modes:** 1. `together` "Our Tower" (rounds 4, ordered): 5-6 block towers with
one friend taking alternating narrated turns. 2. `tall` "Super Tower" (rounds
3, ordered): 7-block towers with two cast friends rotating narrated turns.
**Assets needed:** wooden block art; per-cast turn lines; optional cast turn
portrait presentation.
