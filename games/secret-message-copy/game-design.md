# Secret Message Copy - Game Design

### Mini-GDD: Secret Message Copy (games/secret-message-copy/)
**Category:** writing-fine-motor · **Engine:** build-assemble · **splashEmoji:** 🕵️‍♀️
**Premise:** a secret message arrives as a short word, and the player rebuilds it letter by letter to send it back. The current beta uses the engine's ghost slots as the copy target.
**Learning:** letter-by-letter copying and left-to-right sequencing.
**Modes:** 1. `secret` "Copy the Code" (rounds 5, ordered): CAT, SUN, DOG, HAT, BUS as text letter parts. 2. `long` "Longer Codes" (rounds 3): FROG, CAKE, FISH as text letter parts.
**Assets needed:** spy/envelope frame art; recorded spelling lines.
