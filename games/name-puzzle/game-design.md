### Mini-GDD: Name Puzzle (games/name-puzzle/)
**Category:** writing-fine-motor · **Engine:** sequence-order · **splashEmoji:** 🧩
**Premise:** the QLOBE Kids cast had their names jumbled! Put the letters back in
order. Uses `text:` letter cards; the voice spells along and cheers the name.
**Learning:** letter order, name recognition, left-to-right convention.
**Modes:** 1. `friends` "Fix the Names" (rounds 5, sets = MAYA, LEO, NIA, SAM,
RAVI — items are single letters as `text:M` `text:A`… in order; per-set say line
spells then says the name, e.g. "M-A-Y-A. Maya!"; prompt asks "Can you fix
Maya's name?" — set-level prompt spoken at round start via the set's own
`prompt` if the engine supports it, else the mode prompt stays generic).
2. `words` "Fix the Words" (rounds 4, difficultyRamp, 3-letter words from the
platform word list: CAT, SUN, DOG, BUS as text letters).
**Assets needed:** cast name-card art; recorded spelling voice lines.
