# Sound Cylinder Match Assets

All current art is identical emoji placeholder art rendered by the `match-pairs` engine with `emoji:🥫`. This is intentional for the sound-only matching mechanic. No image files, generated art, downloaded art, recorded clips, or network assets are added by this game.
