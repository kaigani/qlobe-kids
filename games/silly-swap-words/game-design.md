# Silly Swap Words - Game Design

### Mini-GDD: Silly Swap Words (games/silly-swap-words/)
**Category:** reading-phonics · **Engine:** build-assemble · **splashEmoji:** 🔄
**Premise:** cat -> hat -> hot! Each round rebuilds the word with one tile swapped. The ghost slots show the new word's parts, and the voice celebrates the transformation ("Swap the C for an H. HAT! The cat became a hat!").
**Learning:** phoneme substitution - one sound changes the whole word.
**Modes:** 1. `swap` "Swap a Sound" (rounds 5, ordered): chains via onset swaps using shared letter tiles c+at -> h+at, s+un -> b+un, d+og -> l+og, p+ig -> w+ig, m+op -> t+op. 2. `chain` "Word Chains" (rounds 3): cat -> hat -> ham, using verified `at` and `am` rime tiles.
**Assets needed:** swap-arrow animation art; recorded transformation lines.
