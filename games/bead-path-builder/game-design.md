# Bead Path Builder - Game Design

### Mini-GDD: Bead Path Builder (games/bead-path-builder/)
**Category:** writing-fine-motor · **Engine:** build-assemble · **splashEmoji:** 🧵
**Premise:** string beads onto the cord left to right. Each bead snaps to the next ghost spot along a gently curving path. Precision placement feels like a craft activity.
**Learning:** left-to-right motor patterning and controlled placement.
**Modes:** 1. `string` "String the Beads" (rounds 4, ordered): 4-6 bead builds along a curve with swatch and emoji bead parts. 2. `fancy` "Fancy Strings" (rounds 3): alternating shapes such as red-star-red-star along an S-like curve. Repeated fancy beads use visual matching so identical beads cannot produce hidden wrong choices.
**Assets needed:** cord + bead art; voice lines.
