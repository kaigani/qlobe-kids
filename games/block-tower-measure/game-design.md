# Block Tower Measure — Game Design

### Mini-GDD: Block Tower Measure (games/block-tower-measure/)
**Category:** math-number-sense · **Engine:** build-assemble · **splashEmoji:** 🗼
**Premise:** stack the tower — biggest block first! Ordered building teaches
size comparison; the finished tower gets measured aloud ("One, two, three,
four blocks tall!").
**Learning:** size ordering, stacking logic, counting height.
**Modes:** 1. `stack` "Stack It Tall" (rounds 4, ordered): towers of 3-5 blocks,
sizes descending (emoji:🟥 parts with decreasing size values), say counts the
height. 2. `compare` "Two Towers" (rounds 3): build a short (2) and tall (4)
tower side by side (ghost outlines for both), say compares ("The red tower is
TALLER!").
**Assets needed:** wooden block art; voice lines.
