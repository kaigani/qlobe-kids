# Bear — puppet rig character sheet

The "Forest Friends" bear puppet for **Puppet Retell**, rigged for the segmented
spline-puppet engine (`shared/js/stage/puppet.js`). See `docs/puppet-rig-spec.md`
for the authoring contract and `rig.json` for the skeleton/clips.

## Parts

`parts/*.png` — 10 segmented parts on a 1024² transparent canvas, A-pose:
head, torso (green onesie baked in), arm-upper/arm-lower (+paw) ×2,
leg-upper/leg-lower (+foot) ×2. Each keeps full-canvas registration with joint
overlaps (shoulder/elbow/hip/knee caps) so rotation never opens a seam.

## Provenance

- Source art: `00-reference/puppet parts/puppet-parts.png` (A-pose part sheet) and
  `puppet-assembly.png` (annotated joint reference), generated by the author with
  a foundational image model on 2026-07-14 following `docs/puppet-rig-spec.md`.
- Processing: alpha already transparent; parts sliced by connected-component
  bounding box (Pillow/scipy) with 8px pad. Joint coordinates recovered from the
  colored joint pins in `puppet-assembly.png`, scaled to the 1024 canvas, then
  hand-tuned so the built-in overlaps meet in a neutral A-pose.
- License: original assets CC BY 4.0 (consistent with the repo); code MIT.

## Lip-sync — viseme head swap (done)

Full sculpted viseme HEADS (not mouth-patch overlays):
`anim/head-{a,o,e,wr,ts,ln,uq,mbp,fv}.png` (512², transparent). The `head` part
in `rig.json` carries a `visemes` map; the engine swaps the head sprite's texture
via `puppet.setViseme(key)`, driven by `mouth.js` `followCues` against an audio
clip (Rhubarb A–H+X → these viseme names). `ts` is rest/silence. Test harness:
`shared/js/stage/puppet-lipsync.test.html` (verified 12/12 in sync on
`intro-maya`). Bear's own voice clips + Rhubarb cues are a later content step.

Provenance: viseme sheet generated by the author with a foundational image model
(`00-reference/puppet parts/bear/visemes.png`, 3×3 on grey with labels). Prep:
sliced to nine label-free 1024² #919191 tiles (`viseme-tiles/`), then background
separated to transparent via the local `qwen-image-layered` workflow
(`viseme-cutouts/`, layer_2), downsized to 512² into `anim/`. The redraws stay
mutually registered (tops/eyes aligned; only the jaw moves).
